# Audit Finding Corrective Action

## Scenario

An external auditor identified delays in removing terminated employees from systems.

## Deliverable

- Root cause analysis
- Corrective action plan
- Ownership assignment

## Skills Demonstrated

- Audit Support
- Corrective Action Management
- Root Cause Analysis
- Process Documentation

## Framework Alignment

- ISO 27001
- Access Control
- User Lifecycle Management

## Finding

Terminated employees were not consistently removed from systems within 24 hours.

## Lessons Learned

- Root causes should explain why deficiencies exist.
- Monitoring controls are important.
- Accountability should be assigned to responsible teams.
